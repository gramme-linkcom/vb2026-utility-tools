# vb2026-utility-tools

VB2026年環境で開発された、カメラ処理をはじめとする基本機能を搭載したGUIツールアプリケーションです。

---

## 動作環境・開発環境

### 開発環境
- **統合開発環境 (IDE):** Visual Studio 2026
- **ビルドシステム:** CMake
- **パッケージマネージャー:** vcpkg

### 外部ライブラリ
- **OpenCV**
- **Qt5**

---

## 依存ライブラリ（vcpkg）のインストール方法

1. **vcpkg本体の準備**
   vcpkg本体のインストール方法については、下記の解説URLを参照してください。
   → [vcpkgのインストール方法解説（外部サイト）](https://sksp-tech.net/archives/544#toc2)

2. **必須ライブラリの一括インストール**
   本プロジェクトのルートディレクトリで以下のコマンドを実行すると、必要なライブラリが自動的にインストールされます。
   ```bash
   vcpkg install
   ```
