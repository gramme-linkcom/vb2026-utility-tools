# vb2026-utility-tools
VB2026年で開発された、VBのカメラ処理システムなどの基本処理を搭載した、GUIツールです。

# vcpkgのインストール方法
vcpkg本体のインストール方法は、下記のURLを参照してください。
https://sksp-tech.net/archives/544#toc12

インストールが完了したら、下記のコマンドを実行して、必要なライブラリをインストールしてください。
```
vcpkg install
```

なお、上記コマンドを実行すると同時に、Qt5のビルドも開始されます。

Qt5のビルドには、時間がかかることに留意してください。

(詳しくは、『ビルドに関して』の章を参照)

# ビルドに関して
開発当時は、Visual Studio 2026を使用して開発していました。
Cmakeプロジェクトにて構築されています。
外部ライブラリの導入には、vcpkgを使用しています。
外部ライブラリは下記のとおりです。
- OpenCV
- Qt5

Qt5は、ビルドされています。個別でビルドする場合、結構時間がかかることに留意してください。

なお、
- CPU: Intel Core i7-12700K
- RAM: 32GB
- OS : Windows 11 Pro

の環境において、Qt5のビルドにはおおよそ1時間程度かかりました。

(詳細には、57分程度)
